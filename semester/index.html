Semester courses

Hub Hero provides JupyterHubs for those who are teaching introductory programming, statistics or data-science for a whole semester or longer. All your students will have the same programming environment and datasets available to them without the hassle of setting up individual computers. Start teaching from the first minute.

The platform

Hub Hero provides JupyterHubs for teaching professionals. No matter if your students are professionals, high school or university students. It allows you to start with the content of your course on day one instead of having to get everyone setup first. Students can use the platform at anytime, from anywhere.

All your students will have the same environment, libraries and datasets available to them. The platform supports Python, R Studio and C++ out of the box.

Based on the widely used, popular and open-source Jupyter technology all your students need is a web browser. While you could set it up yourself, we offer you peace of mind and our experience in operating these powerful tools reliably.

This offers is ideal for courses that last a few weeks. For details please email semesterhub@wildtreetech.com.

The tools

You and your students will have access to Python notebooks, RStudio and C++ notebooks. The cloud environment for each language comes pre-loaded with the most popular libraries in that language.

The Python environment comes with JupyterLab, Numpy, Scipy, matplotlib, scikit-learn and many more pre-installed.

The RStudio environment comes with the Tidyverse pre-installed.

Features

Student logins can be managed via usernames&passwords or integrated with Google, GitHub or your LMS. Each student will have a home directory for storing their work, data and course materials. Instructors can manage student's environments and course materials can be automatically updated from a central course GitHub repository as the course progresses.
